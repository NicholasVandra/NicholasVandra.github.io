# Employee Attrition Analysis

This project investigates employee attrition using machine learning models,
including **K-Nearest Neighbors (KNN)** and **Naive Bayes (NB)**.
The analysis includes data cleaning, EDA, model comparison, and interpretation
of factors related to employee turnover.
