# PSC Doctor Locator

This project was developed as part of an internship.  
The goal was to create a simple, user-friendly tool that helps users find relevant service providers in their area.

## Features
- Location-based search functionality  
- Interactive map display  
- Ability to filter results based on user needs  
- Clean and responsive UI built with standard web technologies
